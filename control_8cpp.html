Detailed Description

GUI client program to control and display info for with the robot.

Author
Bill
Johnston
Luke Simmons
Date
2022-2-18

This program will connect to the robot via the internet in order to display information about the robot and allow the user to control the robot. The interface is written in GTK and uses the Gtkmm library. It allows the usage of controllers and joysticks on the client to control the robot by using the SDL library. In order to connect to the robot, the user must enter the IP address of the robot in the GUI. It will display information about the voltages and motors on the robot. Define "DEBUG" to print debug information.
